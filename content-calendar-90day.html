Internal Linking Strategy

Rules for every new post

  • Each blog post must link to services.html with anchor text containing a service keyword (e.g. "anxiety therapy", "OCD counseling").
  • City-specific posts must link back to the relevant landing page (therapy-lahore.html / therapy-karachi.html / online-therapy-uk.html).
  • Each post must include an "Also Read" block with 3 internal links to related posts — use the existing block format already on all blog pages.
  • New posts about OCD → always link to blog-cbt-ocd.html. New posts about anxiety → always link to blog-cbt-anxiety.html. Relationship posts → link to blog-relationships-therapy.html.
  • Every post ends with a WhatsApp CTA button with a message pre-filled to match the topic (e.g. "I read your article on OCD and would like to talk").

Blog Post SEO Template (Apply to Every New Post)

Required elements for each post

  • Title tag: ≤65 characters, lead with primary keyword, end with "| Healing with Attia".
  • Meta description: ≤160 characters, include keyword + emotional hook + call to action signal.
  • H1: Contains primary keyword, different from title tag.
  • H2 subheadings: 3–5 per post, include secondary keywords naturally.
  • Stat box: Use the existing .stat-box component for a key insight — increases dwell time.
  • FAQ section: 4 questions minimum with FAQPage JSON-LD schema for rich result eligibility.
  • BlogPosting schema: author "Attia Altaf" (not Dr.), datePublished, image, description.
  • BreadcrumbList schema: Home → Blog → [Post Title].
  • Image alt text: Descriptive, no "Dr." prefix, no "psychologist" label.
  • Word count: 900–1,400 words — enough for topical authority, not padded.
